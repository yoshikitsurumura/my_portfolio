主な実績

金融サロン向けDiscord AIチャットボット

Gemini API × Google Apps Script × Discordで構築。35個の金融ナレッジベースを整備し、投資助言防止のガードレール機能を実装。月額100円の低コストで24時間稼働、自動回答率90%を達成。

Google / Microsoft 365統合ツール開発

AIエージェントプラットフォーム「Patto AI」向けに、Google Workspace（Calendar/Docs/Sheets/Drive/Gmail）とMicrosoft 365（Outlook/OneDrive/SharePoint/Teams）の統合機能を実装。OAuth認証フロー、TypeScript + Bun + Honoのモノレポ構成で開発。

採用管理システム & 業務自動化ツール

CSV + Pythonによる採用管理システム構築。入出力スキーマ定義、バリデーション、集計、レポート生成まで一括化。教育業界では、Excel（VBA/マクロ）を用いたDX推進で生徒数126%、季節講習売上202%向上を実現。

現場の課題解決

教育現場での経験から、日々のルーティン業務に追われ、本来集中すべき「人」との関わりに時間を割けない現状を目の当たりにしてきました。AIを活用し、非効率を解消します。

創造性の解放
