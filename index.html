主要プロジェクト実績

個人病院向け OCR自動処理システム

Python FastAPI Gemini 2.5 Pro Docker

公費受給者証の情報をカメラで読み取り、自動でデータベース化するシステムを構築。Gemini APIを用いた高度なプロンプトエンジニアリングにより、複雑なレイアウトの非定型帳票でも高精度な抽出を実現。医療現場の事務作業負担を大幅に削減。

金融サロン向け Discord AI Bot

Node.js Gemini Flash 2.5 Discord API

投資サロン内での質問に自動回答するAI Botを開発。35件以上の専門ナレッジベースを整備し、投資助言に抵触しないよう厳格なガードレールを実装。月額100円以下の低コスト運用で、コミュニティ運営を24時間サポート。

Google / Microsoft 統合連携モジュール

TypeScript OAuth 2.0 Google/MS Graph API

AIエージェントサービス「Patto AI」向けの共通連携機能を実装。Gmail, Calendar, Drive, Outlook, Teams, OneDrive等へのOAuth認証フローとAPI接続を単一モジュール化。エラーハンドリングやリトライ処理も共通化し、開発効率を向上。
